# healthcare-analytics-dashboard-powerbi
Interactive Healthcare Analytics Dashboard built with Power BI to analyze patient demographics, admissions, billing performance, medical conditions, and test results.

## Project Overview

This project presents an interactive Healthcare Analytics Dashboard built in Power BI to analyze patient demographics, admission trends, medical conditions, billing performance, and test results.

## KPIs
- Total Patients
- Total Billing
- Average Age
- Conditions Treated
- Emergency Cases

## Key Insights
- Admissions peaked between 2020–2023.
- Emergency admissions account for approximately 33% of all admissions.
